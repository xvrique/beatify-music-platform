# Beatify — Modern Music Platform

A digital music streaming interface built with a focus on high-fidelity user experience and modern aesthetics.

## 🚀 Features
- **Dynamic Audio Player:** Interactive controls and playlist management.
- **Responsive Interface:** Seamless experience across mobile and desktop.
- **Sleek UI:** Dark-themed design optimized for music enthusiasts.

## 🛠️ Tech Stack
- Next.js
- EJS
- Tailwind
- Spotify API
